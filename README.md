# Feiyang Chen — Portfolio

A minimal, professional personal portfolio built with Next.js 14 and Tailwind CSS.

## Sections

- **Hero** — Name, title, short intro, and primary CTAs (Get in touch, Download Resume)
- **About Me** — Brief background and focus areas
- **Skills** — Programming languages and tools/methods
- **Projects** — Technical projects with descriptions and tech stacks
- **Research & Experience** — Allada Lab and UM Engineering roles
- **Contact** — Email, phone, LinkedIn, GitHub

## Run locally

**If `npm` is not found**, install Node.js first (macOS with Homebrew):

```bash
brew install node
```

Then:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Resume

Place your PDF at `public/resume.pdf` for the “Download Resume” button. A copy of your CV has been added there from the path you provided.

## Build

```bash
npm run build
npm start
```
